# dto 文件夹说明

本文件夹包含所有数据传输对象（DTO），用于接口参数接收、响应封装和数据校验，是控制器与前端/客户端交互的载体。

---

## 1. ApiResponse.java
- **作用**：统一接口响应格式，封装成功/失败、消息、数据等。
- **关键实现**：静态方法 `ok()`、`error()`，泛型支持任意数据类型。

## 2. LoginRequest.java
- **作用**：登录请求参数对象。
- **关键实现**：包含用户名/手机号、密码、验证码、登录方式等字段，配合 `@Valid` 校验。

## 3. RegisterRequest.java
- **作用**：注册请求参数对象。
- **关键实现**：包含用户名、密码、手机号等注册信息，配合参数校验。

## 4. UserUpdateRequest.java
- **作用**：用户信息更新参数对象。
- **关键实现**：包含可修改的用户字段，如昵称、头像、角色等。

## 5. OrderCreateRequest.java
- **作用**：订单创建参数对象。
- **关键实现**：包含订单明细、收货人、地址、商品列表等，支持嵌套 `OrderItemRequest`。

## 6. OrderItemRequest.java
- **作用**：订单明细项参数对象。
- **关键实现**：包含商品ID、数量、单价等，作为订单创建/修改的子对象。

## 7. ProductRequest.java
- **作用**：商品创建/修改参数对象。
- **关键实现**：包含商品名称、价格、库存、图片等字段。

---

所有 DTO 均采用简单 POJO 结构，结合注解实现参数校验和文档自动生成，保证接口数据安全、规范和易维护。
